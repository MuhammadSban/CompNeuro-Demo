Brain Heatmap — Rendering Demos

Two approaches for mapping sparse electrode values onto a continuous brain surface. Neither demo touches the original project files.

Approach 1

Vertex Coloring

IDW interpolation computed in JavaScript per-vertex, stored in a BufferAttribute. GPU handles interpolation across faces. Simple and fast to set up.

Approach 2

Fragment Shader

Electrode positions + values packed into a DataTexture. IDW and viridis colormap computed per-fragment in GLSL. Resolution-independent, smooth gradients.
